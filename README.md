## Installation

-   `git clone https://github.com/taslimaGit/IT-PRODUCT-INVENTORY-MANAGEMENT`
-   `cd IT-PRODUCT-INVENTORY-MANAGEMENT/`
-   `composer install`
-   `cp .env.example .env`
-   Update `.env` and set your database credentials
-   `php artisan key:generate`
-   `php artisan migrate`
-   `php artisan db:seed`
-   `npm install`
-   `npm run dev`
-   `php artisan serve`
